# Тестовое задание на позицию WebXR разработчика в отдел Робототехники

## Использовавшийся стэк: 
* React
* Three.js
* React Three XR
* Ract Three Fiber **(R3F)**
* Drei
* Vite

## Задачи проекта:
1. Реализовать сцену с освещением
2. Реализовать захват движения VR гарнитуры для движения камеры
3. Загрузить 5 разных 3D моделей в сцену (GLTF/GLB)
4. Реализовать лазерный указатель (луч с маркером на конце)
5. Реализовать подсветку объектов при наведении лазерного указателя
6. Реализовать перемещение объектов с помощью лазерного указателя при взаимодействии с кнопками VR контроллера/ геймпада
7. Реализовать удаление объектов по кнопке VR контроллера/ геймпада
8. Реализовать HTML информационную панель для отображения свойств объектов, а также для добавления новых объектов на сцену
